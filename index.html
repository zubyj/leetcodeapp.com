FAQ

First, make sure you're using the new dynamic Leetcode layout. Leetcode problems will now have company tags in the description tag. The solutions tab will have code solutions and top youtube solution videos, such as Neetcode solutions.

Leetcode Explained reads the title of the tab to determine which problem you are on. This is necessary to display the correct company tags and solution videos in each problem.

No data is collected by Leetcode Explained. All relevant data is stored locally in your browser. If you would like to view the source code, you can find it on our Github.

Yes, Leetcode Explained is entirely free to use. We believe in providing quality resources that can aid in improving coding skills, and we are delighted to offer this to our users free of charge.

Leetcode Explained is an open source project, and we welcome contributions from the community. If you would like to contribute, please visit our Github page.
